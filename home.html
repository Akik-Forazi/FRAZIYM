Omnikarai

A hand-written programming language with a compiler that emits native Windows x64 machine code directly. No LLVM. No GCC backend. No intermediate representation. Every byte of the output binary is authored by the compiler — written entirely in C from scratch.

✓Full Pratt parser + INDENT/DEDENT lexer — Python-like syntax, no braces
✓Direct x86-64 codegen — stack frames, backpatching, Win32 calls, PE binary emitter
✓Functions, recursion, while/for loops, match/case, break/continue, Win32 stdlib
✓v6.0 benchmarks: Fib(40)×5 at 3.72× of GCC -O3, Primes×30 at 3.15× of GCC -O3
✓20/20 benchmark tests passing — correct output verified on every run
